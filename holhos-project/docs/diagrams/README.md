# Diagrams

This folder contains Mermaid diagrams generated from the current backend source code.

- `database-erd.md`: ERD of the SQLAlchemy models / DB schema.
- `backend-class-model.md`: High-level class model (ORM models + Pydantic schemas + services).

How to view:
- VS Code: install a Mermaid preview extension, or use the Markdown preview (some setups render Mermaid blocks).
- GitHub: Mermaid blocks render in Markdown in supported contexts.
