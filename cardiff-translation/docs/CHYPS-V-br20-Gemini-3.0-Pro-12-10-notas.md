# Notas do Neurofisiologista para a Validação:

* **Adaptação Lexical:** O termo "Stripey" foi traduzido como "Listrado" e "Window shade" (no contexto de dirigir) foi adaptado para "Para-sol (ou persiana)" para melhor compreensão local, dado que o uso de "cortina" em carros é menos comum no Brasil para o motorista.
* **Contexto de Desconforto:** A distinção entre desconforto físico e emocional ("Upsetting") foi mantida com o termo "Perturbador", essencial para evitar falsos positivos relacionados a fobias ou aversões não sensoriais.
