# CHYPS-V Scoring

- Responses are scored on a scale from 0-3 (where Almost Never = 0, Occasionally = 1, Often = 2,  Almost Always = 3).
  
- A total score CHYPS-V can be calculated as a sum.

- Separate sum scores can also be calculated for the following subscales:
