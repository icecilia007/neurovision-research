# Escala de Hipersensibilidadee Cardiff - Visual (CHYPS-V)  

## Adaptações Realizadas  

1. **Terminologia técnica**  

   - "Hypersensitivity" → "Hipersensibilidade" (termo padrão em neurofisiologia)  
   - "Strobing lights" → "Luzes estroboscópicas" (jargão científico)  
   - "Shaky camera footage" → "Câmera instável" (termo cinematográfico comum)  

2. **Acessibilidade**  

   - Exemplos contextualizados (ex.: "fazer compras em supermercados" para substituir "supermarkets")  
   - Expressões de intensidade ajustadas ("Almost Always" → "Quase Sempre")  

3. **Formatação**  
   - Mantida estrutura original com títulos, instruções e itens numerados  
   - Opções de resposta alinhadas visualmente com o original  

**Validação Sugerida**:  

- Teste piloto com população brasileira (neurotípica e com TEA/TDAH)
